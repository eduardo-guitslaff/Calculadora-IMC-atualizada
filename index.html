FAQ - Perguntas Frequentes

  • O que é IMC? O IMC (Índice de Massa Corporal) é uma medida usada para avaliar o peso em relação à altura de uma pessoa. Ele é usado para identificar possíveis problemas de peso, como obesidade ou desnutrição.
  • Como o IMC é calculado? O IMC é calculado dividindo o peso da pessoa (em kg) pela altura ao quadrado (em metros). A fórmula é: IMC = peso / (altura * altura).
  • Qual é a faixa normal de IMC? O IMC considerado saudável está entre 18,5 e 24,9. Acima disso, é considerado sobrepeso, e abaixo disso, abaixo do peso.
  • O IMC é aplicável para todas as pessoas? Não, o IMC não é uma medida precisa para todos os tipos de corpo. Ele não leva em consideração a massa muscular ou a distribuição de gordura.
